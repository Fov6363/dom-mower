# DOM Mower / 网页割草机

[English](README.md) | 简体中文

[在线试玩](https://fov6363.github.io/dom-mower/)

把当前网页变成一局轻量割草游戏：网页元素是敌人，页面结构就是关卡。

DOM Mower 是一个无需构建、可直接在本地安装的 Chrome Manifest V3 扩展。所有游戏逻辑都在设备本地运行，不需要账号、服务器或网络权限，也不会读取网页的文字内容。

## 下载

[下载 DOM Mower v0.3.0（ZIP）](https://github.com/Fov6363/dom-mower/releases/download/v0.3.0/dom-mower-v0.3.0.zip)

安装前需要先解压 ZIP。Chrome 无法直接安装普通 ZIP，需要通过“加载已解压的扩展程序”进行安装。

## 安装与试玩

1. 下载并解压上面的 ZIP。
2. 在 Chrome 中打开 `chrome://extensions`。
3. 打开右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择解压后的 `dom-mower-v0.3.0` 目录。
5. 打开一个普通网页，建议先从 Wikipedia 文章页开始。
6. 点击 Chrome 工具栏中的 DOM Mower 图标开始游戏。

再次点击扩展图标、点击右上角的“退出游戏”，或按 `Esc`，即可结束游戏并恢复网页。

## 当前玩法

- 使用 `WASD` 或方向键移动，也可以点击页面，让角色向目标位置移动。
- 贯穿子弹会自动攻击最近的网页元素，三片旋转刀片负责近距离伤害。
- 按 `1` 或 `⌘/Ctrl+F` 使用“查找”，对相同类型的 DOM 元素发动连锁攻击。
- 按 `2` 或 `⌘/Ctrl+X` 使用“剪切”，将附近元素收入剪贴板。
- 按 `3` 或 `⌘/Ctrl+V` 使用“粘贴”，把收集的网页碎片转化为贯穿或爆炸攻击。
- 击碎元素会获得经验。升级时可以选择 `a`、`img`、`>`、`+`、`.class` 和 `!important` 等 CSS/DOM 规则。
- 按钮会向玩家发射红色子弹。
- 清除文字、表单控件和图片后，带有背景、边框或阴影的大型面板会变成精英敌人。
- 敌人归零后，游戏会再次扫描当前可视区域，确认没有遗漏后显示“下一关”。
- 点击“下一关”会向下滚动约 80% 的屏幕高度并生成新一波敌人，到达页面底部即完成本局游戏。

## 权限与隐私

DOM Mower 只申请以下权限：

- `activeTab`：用户点击扩展图标后，读取当前标签页中可见元素的位置和类型。
- `scripting`：向当前标签页注入游戏逻辑和样式。
- `storage`：保存本地游戏进度和设置。

扩展不会申请对所有网站的常驻访问权限，也不包含网络请求、统计 SDK 或用户跟踪。退出游戏后，所有临时样式和游戏界面都会从页面中移除。

## 当前限制

- 只支持页面主滚动条，不支持网页内部的独立滚动容器。
- 不读取文字内容，只根据元素标签、位置和尺寸生成敌人。
- 不支持跨域 iframe、Shadow DOM、视频和 Canvas 元素。
- Chrome 不允许扩展在 `chrome://` 等浏览器内部页面中运行。
- 某些网页使用激进的 CSS 动画时，受击反馈可能与原有效果冲突；退出游戏时仍会移除全部临时 class。

## 开发

项目使用原生 JavaScript、Canvas 2D 和 CSS，没有依赖和构建步骤。可以运行 `git clone https://github.com/Fov6363/dom-mower.git` 获取源码。修改后，在 `chrome://extensions` 中重新加载扩展即可。
